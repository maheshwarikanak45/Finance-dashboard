# 💰 Finance Dashboard

A personal finance tracking web app built with React. 
Track your income and expenses, visualise spending patterns 
with charts, and manage transactions — all saved locally in your browser.

## 🔗 Live Demo
https://maheshwarikanak45.github.io/Finance-dashboard/

## ✨ Features
- Add, edit and delete income/expense transactions
- Summary cards showing total balance, income and expenses
- Pie chart showing spending breakdown by category
- Bar chart showing monthly income vs expense comparison
- Search and filter transactions by type, category or title
- Analytics page with spending statistics
- Data persists across sessions using localStorage
- Fully client-side routed with multiple pages

## 🛠️ Tech Stack
- React 19
- React Router DOM
- Tailwind CSS
- Recharts
- date-fns
- Vite
- localStorage
